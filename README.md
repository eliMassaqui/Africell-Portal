# Africell Portal - Contexto do App

O **Africell Portal** é uma **simulação interativa de uma plataforma de usuário** de uma operadora de telecomunicações. Ele foi criado para **dar a sensação de estar em um portal real**, onde você pode interagir com funcionalidades como login, perfil, ranking e quizzes, mas tudo acontece **apenas localmente**, sem backend real.

A ideia do app é **reproduzir a experiência completa de um portal**, oferecendo ao usuário:

- **Sensação de autenticação real**: criação e login de contas fictícias.  
- **Perfil dinâmico**: pontos, níveis e badges que mudam conforme você interage.  
- **Ranking simulado**: competição divertida com outros usuários fictícios.  
- **Quizzes interativos**: aprendizado e diversão combinados, com pontuações simuladas.

O foco principal do portal é **entretenimento e interação**, usando uma interface colorida e moderna inspirada nas cores da Africell, para criar uma experiência **realista e envolvente**, mesmo sem nenhum dado real sendo armazenado no servidor.
